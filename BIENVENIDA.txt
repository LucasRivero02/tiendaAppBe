🚀 QUICK START (15 minutos)
═══════════════════════════════════════════════════════════════════════════════

  PASO 1: Crear .env
  ───────────────────────────────────────────────────────────────────────────
  $ cat > .env << EOF
  MONGODB=mongodb+srv://user:pass@cluster.mongodb.net/tienda
  SECRET_KEY=my_jwt_secret_key_123
  FRONTEND_URL=http://localhost:3000
  EOF

  PASO 2: Instalar dependencias
  ───────────────────────────────────────────────────────────────────────────
  $ npm install

  PASO 3: Testing Local (SIN AWS)
  ───────────────────────────────────────────────────────────────────────────
  $ npm run offline
  ✓ Servidor corriendo en http://localhost:3000/dev
  
  PASO 4: Probar endpoints (en otra terminal)
  ───────────────────────────────────────────────────────────────────────────
  $ curl http://localhost:3000/dev/producto
  $ bash test-lambda.sh  # Testing automático

  PASO 5: Configurar AWS (si quieres ir a producción)
  ───────────────────────────────────────────────────────────────────────────
  $ aws configure
  # Ingresar: Access Key, Secret Key, región, formato

  PASO 6: Desplegar
  ───────────────────────────────────────────────────────────────────────────
  $ npm run deploy -- --stage dev
  ✓ API desplegada en AWS Lambda
  ✓ URL: https://xxx.execute-api.us-east-1.amazonaws.com/dev
